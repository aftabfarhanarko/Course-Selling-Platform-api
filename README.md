# Course Selling Platform API

A modern and scalable backend API for the **Course Selling Platform**, built with **NestJS**, **TypeScript**, **PostgreSQL**, and **Prisma ORM**. The platform provides secure authentication, course management, student enrollment, payment processing, and administrative controls for a complete online learning experience.

## 🚀 Features

* JWT Authentication & Authorization
* Role-Based Access Control
* Course Management
* Student Enrollment System
* Secure REST API
* PostgreSQL Database
* Prisma ORM Integration
* Input Validation
* Error Handling
* Scalable Architecture
* Environment-Based Configuration

## 🛠️ Tech Stack

* NestJS
* TypeScript
* PostgreSQL
* Prisma ORM
* JWT Authentication
* Node.js
* Express.js

## 📦 Installation

```bash
git clone https://github.com/aftabfarhanarko/Course-Selling-Platform-api.git

cd Course-Selling-Platform-api

npm install
```

## ⚙️ Environment Variables

Create a `.env` file:

```env
PORT=5000

DATABASE_URL="postgresql://username:password@localhost:5432/course_selling_platform"

JWT_SECRET=your_jwt_secret
```

## ▶️ Run Project

```bash
npm run start:dev
```

## 🗄️ Prisma Commands

```bash
npx prisma generate

npx prisma migrate dev

npx prisma studio
```
